# roblox_car_game

Rojo is configured with [`default.project.json`](/Users/davidkerschensteiner/Coding/Roblox%20Car/default.project.json) and maps:
- `ReplicatedStorage/`
- `ServerScriptService/`
- `StarterPlayer/`
- `StarterGui/`

## Studio Sync
1. Install Rojo (via Aftman or standalone binary).
2. From this repo root, run:
   ```bash
   rojo serve
   ```
3. In Roblox Studio, open the Rojo plugin and connect to `localhost:34872`.
4. Use Play Solo to test changes live after sync.
